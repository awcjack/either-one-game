# 🎮 二選一淘汰賽遊戲

一個互動式的二選一淘汰賽遊戲，讓你可以自訂選項並進行有趣的比較遊戲。

## ✨ 功能特點

- **📝 自訂編輯器**：可以自由添加、編輯和刪除選項
- **🖼️ 圖片支援**：每個選項都可以上傳自訂圖片
- **🎯 淘汰賽機制**：通過二選一的方式逐步淘汰，直到選出最終勝者
- **🔗 分享功能**：通過 URL 分享你的遊戲設定（圖片和文字都會編碼在連結中）
- **📱 響應式設計**：支援桌面和移動設備

## 🎯 預設選項

遊戲預設包含 13 個選項：

1. 溫婉

## 🚀 使用方法

### 編輯模式

1. 點擊圖片區域上傳自訂圖片
2. 在文字框中輸入或修改選項文字
3. 使用「×」按鈕刪除不需要的選項
4. 點擊「+ 新增選項」添加更多選項

### 開始遊戲

1. 確保至少有 2 個選項
2. 點擊「開始遊戲」按鈕
3. 在每一輪中，點擊你喜歡的選項
4. 被選中的選項會保留，另一個會被淘汰
5. 繼續選擇，直到剩下最後一個選項

### 分享遊戲

1. 在編輯模式下，點擊「分享遊戲」按鈕
2. 系統會生成包含所有選項數據的連結
3. 複製連結分享給朋友
4. 朋友打開連結後，會自動載入你設定的所有選項和圖片

## 🛠️ 技術特點

- **純前端實現**：使用 HTML、CSS 和原生 JavaScript
- **Base64 編碼**：圖片和文字都會編碼到 URL 中
- **無需後端**：所有數據都存儲在 URL 參數中
- **美觀動畫**：流暢的過渡和互動效果

## 💡 技術說明

### URL 編碼機制

遊戲使用以下方式將數據編碼到 URL：

1. 將所有選項（包含文字和 base64 圖片）序列化為 JSON
2. 使用 `btoa()` 和 `encodeURIComponent()` 進行雙重編碼
3. 將編碼後的字符串作為 URL 參數 `data` 傳遞
4. 接收者訪問連結時，自動解碼並載入遊戲設定

### 圖片處理

- 使用 FileReader API 將上傳的圖片轉換為 base64 格式
- Base64 圖片直接存儲在選項數據中
- 分享時整個 base64 圖片會包含在 URL 中
